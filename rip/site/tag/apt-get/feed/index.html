GPG Error NO_PUBKEY

Ultimamente eu tenho encontrado um erro GPG ao executar o apt-get update:

$ sudo apt-get update
W: GPG error: http://ppa.launchpad.net saucy Release: The following signatures couldn't be verified because the public key is not available: NO_PUBKEY 2EE5793634EF4A35

O erro acontece porque não foi possível encontrar a chave de criptografia no sistema. Para corrigir basta adicionar a chave (sequencia de caracteres após NO_PUBKEY).

sudo apt-key adv --recv-keys --keyserver keyserver.ubuntu.com 2EE5793634EF4A35

Agora pode rodar o apt-get update que deve funcionar normalmente.

Normalmente quando você executa o configure ou o próprio make.

Uma dica boa é instalar o apt-file (caso já não tenha instalado) e executar um search pelo pacote que contenha o tal arquivo faltando e assim resolver o problema.

# Instale o apt-file.
sudo apt-get update
sudo apt-get install apt-file

# Atualize o cache do apt-file.
sudo apt-file update

# Procure pelo arquivo.
sudo apt-file search StdSel.h
    libxmu-headers: /usr/include/X11/Xmu/StdSel.h

# Agora instale o pacote que contém o arquivo.
sudo apt-get install libxmu-headers

# Opcionalmente apague o cache do apt-file para liberar espaço em disco.
sudo apt-file purge

Pronto! Simples mas ajuda muito.
