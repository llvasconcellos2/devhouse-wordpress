Para concluir este tutorial você vai precisar acesso a internet, um cd virgem, regravável ou um pendrive e um pouco de paciência.

Então, você que quer deixar o windows de lado e usar um sistema operacional de verdade ou você que já usa Linux mas está de saco cheio com o Ubuntu que tal Linux de Menta?

Vamos lá?

Instalação do Linux Mint Debian Edition 201204


O primeiro passo é fazer o donwload da imagem ISO do cd de instalação do LMDE. Para isso vá ao site www.linuxmint.com, selecione download e Linux Mint Debian ou clique aqui para ir direto. Em important links selecione MATE/Cinnamon para Gnome 2 e 3 respectivamente ou Xfce para o Desktop Xfce. Existem opções de 32 ou 64 bits. No meu caso escolhi Gnome e 64 bits.

Download do Linux Mint Debian Edition.

Na próxima página você pode escolher um mirror mais próximo para fazer download via http ou torrent. Eu escolhi torrent para garantir a integridade do download já que torrent verifica md5 do arquivo.

Escolha um mirror para download.

Com o arquivo iso em mãos você pode gravar um cd ou gerar um pendrive de boot. O meu computador não tem drive ótico então vou precisar do Unetbootin. Um utilitário para gerar pendrives de boot para instalar Linux. Existem versões para windows, linux e mac.

Download do Unetbootin, gerador de pendrive de boot.

Ao executar o Unetbootin selecione “Imagem de disco” ISO e no botão “…” navege até o arquivo que você fez download do LMDE. Selecione a letra da unidade onde está inserido o pendrive e clique em OK.

É importante já ter o pendrive formatado em Fat32 antes para que este passo seja executado com sucesso.

Preparando o pendrive para instalação.

Após a criação do disco de boot, insira o mesmo no seu computador, entre na bios e selecione a opção de ordem de boot para o pendrive ou cd correto. Este passo é importante e depende do modelo de cada BIOS e por isso não é descrito aqui.

Ao ligar o computador com o disco de boot inserido aparecerá a seguinte tela. Selecione default e pressione enter:

Menu de boot do Grub com UnetBootin


O sistema fará boot no livecd e quando estiver pronto clique no ícone Install Linux Mint

Instalar Linux Mint Debian Edition


Escolha a sua lingua. Eu prefiro o Linux em inglês para facilitar a busca de informações na internet mas você pode escolher a lingua que preferir.

Seleção de línguas para instalação


Escolha seu fuso horário:

Escolha seu fuso horário.


Escolha o layout do seu teclado. Teclados com cedilha são português (Brazil) e sem ccedilha você pode escolher: Estados Unidos –  English International.

Escolha o layout do teclado.

Selecione o HD a ser instalado o sistema.

Escolha o HD para instalação

Selecione a partição onde sera instalado o Linux. No meu caso o disco está limpo e por isso as partições deverão ser criadas. Clique em Edit partitions.

Seleção da partição para instalação.

A tela do Gparted abrirá. Clique em Partition -> New.

Criação da nova partição.

No campo Free space following (MiB) preencha o tamanho em megabytes que vai sobrar no disco após a criação da primeira partição. Basicamente você deixa o espaço livre que você usará para a partição de swap. No meu caso vou usar o mesmo valor que tenho de memória. Provavelmente não tem necessidade de tanto mas na dúvida mal não vai fazer. Em File system selecione ext4 e clique em Add.

Deixando espaço para a partição de swap.

Selecione o espaço unallocated, vá no menu Partition -> New. Vamos usar o restante para a partição de swap. Em File system selecione linux-swap. Clique em Add.

Criando a partição de Swap.

Seu disco ficará parecido com a figura abaixo: Vá no menu Edit -> Apply All Operations.

Resultado do particionamento.

Confirme clicando em apply.

Confirmação das alterações.

As partições serão criadas. Feche a janela do Gparted e clique no botão refresh. As partições criadas vão aparecer no instalador do linux. Clique com o botão direito do mouse na partição do tipo ext4 e selecione Assign to /. O mout point da partição primaria ficara em /. Clique em forward.

Definindo o ponto de montagem.

Entre com as informações pedidas e o nome do computador na rede (hostname).

Nome de usuário, senha e hostname.

Na próxima tela deixe o padrão ou escolha uma partição específica para o Grub.

Onde gravar o Grub?

Revise as configurações e clique em install.

Revisão das configurações.

O processo de instalação e cópia dos arquivos será iniciado. Vá tomar café e volte daqui a alguns minutos.

Cópia dos arquivos.

Ao termino da instalação clique em Quit e reinicie o computador. Não esqueça de tirar o pendrive.

Pronto!
