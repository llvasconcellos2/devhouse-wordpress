Acabei de fazer o download do pacote deb do Google Chrome e ao clicar duas vezes no arquivo quem abriu o mesmo foi o File Roller e não o GDebi, que é o instalador de pacotes padrão do Linux Mint Debian Edition 12. Em falar nisso o GDebi foi feito pela Canonical (será que tem Ubuntu no meu Debian? :-P).

Eu prefiro que o arquivos .deb fiquem associados com o GDebi para facilitar a vida e para fazer isso basta clicar com o botão direito do mouse no arquivo e ir para a aba “Open with” em inglês, selecione o GDebi Package Installer e clique em Set as Default.

.deb default para GDeb

Obviamente isso serve para qualquer tipo de arquivo. Caso o programa preferido não apareça entre os programas recomendados clique em “Show other applications”. E se o seu programa não aparecer em “Show other applications”? Você provavelmente vai precisar criar um arquivo .desktop para ele. Veja como fazer aqui.

Observe que prefiro o meu Linux em inglês, apesar de todas as configurações de localização e teclado estarem para Português/Brasil. Prefiro assim pois facilita na hora de pesquisar por dúvidas, mensagens de erro, nomes de programas, etc.

Então é isso! A dica é simples mas eu não conhecia…
