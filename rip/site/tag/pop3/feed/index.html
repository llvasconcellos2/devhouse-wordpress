Error: (state SENT_SECOND_EHLO) code(0)

Ou

Error: [Server response: read error code(0) ]

A solução é simples: altere o endereço do servidor SMTP de outlook.office365.com para o servidor específico para a sua localização geográfica.

  1. Abra o prompt de comando e  digite: ping outlook.office365.com
  2. Anote qual o endereço de retorno como por exemplo: outlook-namsouth.office365.com

  3. Insira outlook-namsouth.office365.com no lugar de outlook.office365.com
  4. Use seu email completo como usuário.
  5. Porta 587 e criptografia TLS.

Pronto!
