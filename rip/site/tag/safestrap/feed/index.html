A ideia é usar o CyanogenMod diariamente para aproveitar os recursos do KitKat mas manter a ROM Stock com Jelly Bean para usar conectado a um monitor.

Para isso você vai precisar fazer o download dos seguintes arquivos:

  • SafeStrap 3.65 para umts_spyder (codenome do Motorola Razr GSM)
  • CyanogemMod (Salvei a versão que usei aqui também: cm-11-20140104-SNAPSHOT-M2-umts_spyder.zip)
  • Google Apps (aplicativos do google que não estão inclusos na CyanogenMod por padrão). Salvei a versão que usei aqui também: gapps-kk-20131208.zip)

Lembrando que você precisa ter root no seu Razr para poder instalar o Safestrap. Caso ainda não tenha siga o tutorial abaixo:

  • Root no Motorola Droid Razr XT910 [Jelly Bean 4.1.2]

    janeiro 8. 201471Comment

    Para executar o exploit que dá acesso ao usuário root no Android do Motorola Droid Razr XT910 você vai precisar dos seguintes...

Siga os seguintes passos:

  1. Copie o arquivos arquivos zip do CyanogemMod e Google Apps para o dispositivo.
  2. Copie o arquivo Safestrap-UMTS_Spyder-3.65.apk para o telefone e instale.
  3. Abra o aplicativo e dê permissão de root.
  4. Selecione a opção: Install Recovery
  5. Desligue o aparelho e torne a ligar.
  6. O menu de Boot do SafeStrap vai aparecer conforme a imagem:

    Opções de Boot do SafeStrap

  7. Selecione [MENU] para entrar no Safestrap. No cabeçalho aparecerá em vermelho a mensagem: ROM Slot: stock que quer dizer que as alterações serão feitas na ROM de fábrica, então vá em Boot Options e selecione ROM-Slot-1

    Boot Options do Safestrap

  8. A tela seguinte exibe as opções de tamanho para a nova imagem do sistema onde será instalada o CyanogenMod. Deixe a System partition size em 700 MB e Cache partition size em 300 MB (valores padrão). Já a opção Data partition size é o tamanho destinado a instalação de aplicativos. Na primeira vez que criei a imagem diminui a partição Data para 1000 MB mas logo o espaço foi preenchido e tive problemas portanto se você usa muitos aplicativos aumente deixe em 2000 MB ou aumente até 4000 MB. Lembre-se que este espaço será ocupado no armazenamento interno do seu Razr.

    Criação de Slot de Rom do SafeStrap

  9. O slot para a ROM será criado:

    Criando ROM Slot

  10. Após o término do processo verifique se o cabeçalho do SafeStrap exibe a mensagem ROM Slot: rom-slot1. Caso ainda esteja selecionada a ROM Stock vá em Boot Options – ROM-Slot-1 – Activate.

    ROM Slot 1 ativo no SafeStrap

  11. No menu principal do SafeStrap selecione Install.

    Menu principal do SafeStrap

  12. Será exibida uma tela para seleção do arquivo zip a ser instalado. Navegue até o diretório onde você copiou os arquivos zip e selecione primeiro o arquivo da CyanogenMod (cm-11-xxxx.zip) depois vá em Add More Zips e selecione o arquivo zip do Google Apps (gapps-xxx.zip).

    Seleção de Zip para Instalação do SafeStrap

  13. Faça o swipe para confirmar a instalação (Swipe to confirm flash). No meu aparelho a instalação do GApps falha mas ignorei a mensagem e ao bootar a ROM e usar o Google Play pela primeira vez ocorre um erro por falta de um componente que acredito que o Play instala automaticamente pois a mensagem não se repetiu e tudo funcionou bem. Provavelmente isso ocorre pois a CM11 no Razr até o presente momento não é estável e ainda está em desenvolvimento. Inclusive se alguém experimentar algum bug peço que compartilhe aqui nos comentários para ficarmos por dentro.

    Swipe para confirmar a instalação

  14. O processo de instalação iniciará:

    Instalando zips usando o SafeStrap

  15. Ao finalizar a instalação selecione a opção Wipe cache/dalvik e depois vá ao menu principal e selecione Reboot – System para reiniciar para o seu Novo Android 4.4 KitKat!

    Animação de Boot do CyanogenMod 11 KitKat

Depois você pode alterar entre as ROM Slots ativas para selecionar qual sistema dar boot.
