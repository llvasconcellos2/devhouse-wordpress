Client Side Image Resize

Client Side Image Resize

Resize large images in the browser before uploading to WordPress — save time and bandwidth.

Legacy plugin — kept here for reference

What it does

This plugin uses HTML5 to resize large images directly in the browser before they’re uploaded to WordPress — no extra software needed, even for high-resolution camera photos.

  • ✅ First plugin compatible with WordPress 3.9
  • ✅ Works with drag & drop uploads
  • ✅ Adds settings under Settings → Media to control resized JPG quality, max width and max height

Compatible browsers: Chrome, Firefox, Opera, iOS and Android.

Installation

  1. Upload the client-side-image-resize folder to /wp-content/plugins/.
  2. Activate the plugin from the Plugins menu in WordPress.

Client Side Image Resize (Português)

Com este plugin você pode usar diretamente as imagens em alta resolução da sua câmera digital no WordPress, sem precisar de softwares adicionais para redimensioná-las. Basta ter um navegador atualizado.

  • ✅ Primeiro plugin compatível com WordPress 3.9
  • ✅ Funciona com arrastar e soltar (drag & drop)
  • ✅ Adiciona configurações em Configurações → Mídia para qualidade do JPG e largura/altura máximas

Navegadores compatíveis: Chrome, Firefox, Opera, iOS e Android.

Instalação

  1. Faça o upload da pasta client-side-image-resize para /wp-content/plugins/.
  2. Ative o plugin pelo menu Plugins do WordPress.
