Normalmente quando você executa o configure ou o próprio make.

Uma dica boa é instalar o apt-file (caso já não tenha instalado) e executar um search pelo pacote que contenha o tal arquivo faltando e assim resolver o problema.

# Instale o apt-file.
sudo apt-get update
sudo apt-get install apt-file

# Atualize o cache do apt-file.
sudo apt-file update

# Procure pelo arquivo.
sudo apt-file search StdSel.h
    libxmu-headers: /usr/include/X11/Xmu/StdSel.h

# Agora instale o pacote que contém o arquivo.
sudo apt-get install libxmu-headers

# Opcionalmente apague o cache do apt-file para liberar espaço em disco.
sudo apt-file purge

Pronto! Simples mas ajuda muito.
