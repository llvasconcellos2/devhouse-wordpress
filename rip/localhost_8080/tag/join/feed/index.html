É possível fazer um UPDATE com INNER JOIN Veja abaixo:

UPDATE
    tabela_1
INNER JOIN
    tabela_2 ON tabela_2.tabela_1.id = tabela_1.id
SET
    tabela_1.nome_do_campo = tabela_2.nome_do_campo

Simples!
