Como já falei em posts anteriores uso o Linux Mint Debian Edition, que nada mais é do que o Debian Testing com alguns repositórios do Mint e por isso não existe disponível ainda o gimp 2.8 nos repositórios do Mint mas já está disponível no Debian Testing e portanto vamos usá-lo.

Primeiro remova a versão antiga

sudo apt-get purge gimp*

Agora adicione o repositório do Debian Testing.

sudo add-apt-repository "deb http://ftp.debian.org/debian/ testing main contrib non-free"

E finalmente instale o Gimp

sudo apt-get update && sudo apt-get install gimp

Caso apresente problemas de dependências use o aptitude e escolha a opção onde todos os pacotes são atualizados para corrigir o problema. No meu caso foi a terceira opção.

sudo aptitude install gimp
